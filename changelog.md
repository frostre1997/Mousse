# Changelog

## v1.0.0 (Initial Release)

- Custom mouse cursor rendered on screen
- Mouse move tracking via JNI bridge
- Left click support (tap/select/jump)
- Right click support (secondary action / delete in editor)
- Scroll wheel support (navigation, editor zoom)
- Configurable pointer sensitivity
- Configurable cursor size
- Toggle cursor visibility
- Invert scroll direction option
- Automatic touch suppression when mouse is connected
- Re-parents cursor on scene changes
- Android JNI bridge with native method registration
